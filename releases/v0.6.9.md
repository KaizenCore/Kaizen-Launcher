# Kaizen Launcher v0.6.9

**Release Date:** December 21, 2025

## Highlights

- **Panel-Based Playground** - Complete redesign with 3 resizable panels
- **Quick Add Mods** - Search and install from Modrinth instantly
- **Console Font Controls** - Ctrl+Scroll, slider, and +/- buttons

## What's New

### Panel-Based Playground Layout

The Playground has been completely redesigned from a node-based canvas to a clean 3-panel layout:

- **Left Panel** - Mod list with search, sort, and filter options
- **Center Panel** - Console with real-time logs and server commands
- **Right Panel** - Details with mod info, config editor, and instance settings

All panels are:
- Resizable with drag handles
- Collapsible via toolbar buttons
- Responsive and smooth

### Quick Add Mod Dialog

New dialog to quickly search and install mods from Modrinth:

- Real-time search with debounce
- Shows mod icon, title, author, and download count
- Auto-selects best compatible version
- Infinite scroll for browsing results
- Shows "Installed" badge for already-installed mods

### Import Local Mods

Upload button in the mod list to import .jar files:

- Multi-file selection
- Direct copy to mods folder
- Instant refresh after import

### Console Font Size Controls

Adjust console text size with multiple methods:

- **Ctrl+Scroll** - Zoom in/out with mouse wheel
- **Slider** - Drag to adjust (8px to 20px)
- **+/- Buttons** - Precise adjustments
- Persists across sessions via localStorage

### Visual Config Editor Improvements

Enhanced GUI editor for mod configs:

- Support for JSON, TOML, YAML, and .properties files
- Toggle switches for booleans
- Sliders for numbers
- Color pickers for hex colors
- Nested object and array editing
- Toggle between visual and text modes

### Mod List Enhancements

Better mod management in the left panel:

- Search by mod name or filename
- Sort: name (A-Z, Z-A), enabled first, disabled first
- Filter: all, enabled, disabled, missing dependencies
- Infinite scroll with lazy loading
- Batch selection with "Select All"
- Missing dependency warnings

## Technical Changes

- Removed React Flow dependency and node-based architecture
- New `PlaygroundLayout.tsx` with `react-resizable-panels`
- New `panels/` directory with modular components
- Simplified `playgroundStore.ts` state management
- Added `get_mods_folder_path` Tauri command
